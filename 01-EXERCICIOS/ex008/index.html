Texto marcado / REALCE

Podemos criar também um texto marcado/realçado usando a tag MARK

Você não consegue mudar a cor do REALCE no HTML, tem que ser com CSS, pois trata-se de "forma" e não de conteúdo. Pra isso, dentro de MARK teremos que usar comandos/tags de CSS, no caso STYLE="background-color". A tag CSS tem que estar dentro da MARK. Pra escolher a cor CTRL+ESPAÇO. Dentro do quadrado da cor, passe o mouse pra abrir a barra de cores e poder escolher o tom. Veja neste teste que a palavra TESTE ficou de outra cor.

Pra alterar a cor em todo o projeto, tem que ser na parte de head. Abro uma linha STYLE, com parâmetro MARK e coloco a cor que vai ser aplicada em todo o projeto

Estamos criando um texto grande e um texto pequeno nesse parágrafo.
